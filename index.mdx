AlphaFold v2 predicts protein structures with accuracy competitive with experimental methods. It supports:

## How it works

<Steps>
  <Step title="Prepare your sequence">
    Create a FASTA file with the amino acid sequence(s) you want to fold. For multimers, include all chains in the same file.
  </Step>
  <Step title="Run AlphaFold">
    Use the Docker script to launch a prediction. AlphaFold searches genetic databases, builds MSAs, searches for structural templates, and runs the neural network model.
  </Step>
  <Step title="Analyze outputs">
    The prediction produces ranked PDB structure files, per-residue pLDDT confidence scores, and PAE matrices for assessing domain packing confidence.
  </Step>
</Steps>

<Note>
  AlphaFold requires a Linux machine with an NVIDIA GPU. Full database setup requires approximately 2.62 TB of disk space. See [Installation](/installation) for hardware requirements.
</Note>
